## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

<p>1. Clone repository</p>

```
git clone https://github.com/kyordanov-intellias/task-4-Redux.git
```

<p>2. Open folder</p>

```
cd .\task-4-Redux\
```

<p>3. Install Dependencies</p>

```
npm install
```

<p>4. Run the application</p>

```
npm run dev
```
